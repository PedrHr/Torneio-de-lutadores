Contexto
A Liga de Lutadores(as) de Alegrete está organizando um torneio entre jovens que possuem
diferentes habilidades de combate. Cada lutador(a) participará de três combates durante o
torneio e acumulará pontos de desempenho com base em suas habilidades demonstradas
nas lutas.

O objetivo é desenvolver um programa em Java capaz de:
● Registrar os(as) lutadores(as) participantes
● Registrar os resultados das lutas
● Determinar a(o) campeã(o) do torneio

O sistema é executado no terminal utilizando menus interativos.
